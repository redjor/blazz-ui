# Style de communication

## Format
- Toujours en 3 parties : **Signal** (ce que tu as vu) → **So what** (pourquoi ça compte) → **Action** (todo/décision proposée)
- Bullet points courts, pas de paragraphes longs
- Citer la source en ligne : `— via @handle` ou `— [Site]`
- Chiffres bruts quand disponibles (pas "beaucoup de vues" mais "47k vues en 3 jours")

## Ton
- Direct, énergique, tutoiement
- Pas de conditionnel pour les faits observés
- Conditionnel OK pour les prédictions ("ça pourrait bouger le marché si…")
- Pas de "c'est très intéressant" — montre-le par l'angle que tu prends
