## Prerequis

- Node.js 20+
- PostgreSQL (pour le CRM demo)
- npm ou pnpm

## Installation

```bash
git clone <repo-url>
cd blazz-ui-app
npm install
```

## Configuration

### Variables d'environnement

Creez un fichier `.env` a la racine :

```ini
DATABASE_URL="postgresql://user:password@localhost:5432/blazz_crm"
```

### Base de donnees

```bash
npx prisma db push
npx prisma db seed
```

## Demarrage

```bash
npm run dev
```

L'application demarre sur [http://localhost:3100](http://localhost:3100).

## Structure du projet

```
app/(dashboard)/     → Routes CRM (companies, contacts, deals...)
app/(frame)/         → Showcase UI kit + documentation composants
components/blocks/   → Composants metier (PageHeader, StatsGrid, DataGrid...)
components/ui/       → Primitives shadcn etendu
components/features/ → Composants avances (DataTable, CommandPalette...)
components/layout/   → Layouts et navigation
lib/actions/         → Server Actions (CRUD)
lib/schemas/         → Schemas Zod partages
prisma/              → Schema et seed data
```

## Conventions

- **Server Components par defaut** — Client (`"use client"`) uniquement pour l'interactivite
- **Formulaires** = react-hook-form + zod, toujours
- **4 etats obligatoires** pour chaque composant : loading (Skeleton), empty, error, success
- Pas de prefix `src/` — fichiers a la racine
