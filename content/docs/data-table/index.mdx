## Vue d'ensemble

Le **DataTable** est le composant de table le plus avance du kit. Il est construit sur [TanStack Table](https://tanstack.com/table) et offre :

- **Filtres composables** — filtre builder avec operateurs AND/OR
- **Vues sauvegardees** — enregistrer et charger des configurations de filtres/colonnes
- **Tri multi-colonnes** — tri URL-driven
- **Selection en masse** — checkbox + bulk action bar
- **Actions par ligne** — menu d'actions contextuel
- **Pagination** — serveur-side avec compteurs
- **Colonnes editables** — edition inline des cellules

## Import

```tsx
import { DataTable } from "@/components/features/data-table/data-table"
```

## DataTable vs DataGrid

| Fonctionnalite | DataGrid (blocks) | DataTable (features) |
|----------------|-------------------|---------------------|
| Complexite | Simple | Avance |
| Filtres | Non | Oui (filter builder) |
| Vues | Non | Oui (sauvegardees) |
| Edition inline | Non | Oui |
| Bulk actions | Basique | Complet (bar flottante) |
| Presets CRM | Non | Oui (8 presets) |

Utilisez **DataGrid** pour les tables simples et **DataTable** pour les ecrans data-heavy avec filtres et vues.

## Utilisation rapide

```tsx
import { DataTable } from "@/components/features/data-table/data-table"
import { columns, filterFields } from "@/components/features/data-table/presets/crm-contacts"

export default function ContactsPage({ data, totalCount }) {
  return (
    <DataTable
      columns={columns}
      data={data}
      totalCount={totalCount}
      filterFields={filterFields}
    />
  )
}
```

Pour plus de details, voir [Presets](/docs/data-table/presets) et [Column Factories](/docs/data-table/columns).
