---
title: StatsGrid
description: Grille de statistiques KPI avec icones, tendances et etat de chargement.
---

## Import

```tsx
import { StatsGrid } from "@/components/blocks/stats-grid"
```

## Utilisation

```tsx
import { Users, DollarSign, TrendingUp, Package } from "lucide-react"

const stats = [
  { label: "Contacts", value: "1,234", trend: 12, icon: Users },
  { label: "Revenue", value: "€45,200", trend: 8.5, icon: DollarSign },
  { label: "Deals actifs", value: 47, trend: -3, trendInverted: false, icon: TrendingUp },
  { label: "Produits", value: 156, icon: Package },
]

<StatsGrid stats={stats} columns={4} />
```

## Props

### `StatsGridProps`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `stats` | `StatItem[]` | **requis** | Liste des statistiques a afficher |
| `columns` | `2 \| 3 \| 4` | `4` | Nombre de colonnes sur grand ecran |
| `loading` | `boolean` | `false` | Affiche des squelettes de chargement |
| `className` | `string` | - | Classes CSS additionnelles |

### `StatItem`

| Prop | Type | Description |
|------|------|-------------|
| `label` | `string` | Libelle de la statistique |
| `value` | `string \| number` | Valeur affichee |
| `trend` | `number` | Pourcentage de tendance (positif = hausse, negatif = baisse) |
| `trendInverted` | `boolean` | Inverser la couleur de tendance (baisse = vert, hausse = rouge) |
| `icon` | `LucideIcon` | Icone affichee en haut a droite |

## Variantes

### Etat de chargement

```tsx
<StatsGrid stats={[]} columns={4} loading={true} />
```

Affiche des `Skeleton` dans chaque carte.

### 2 colonnes

```tsx
<StatsGrid stats={stats.slice(0, 2)} columns={2} />
```

### Tendances inversees

Pour des metriques ou une baisse est positive (ex: taux de churn, nombre de bugs) :

```tsx
const stats = [
  { label: "Taux de churn", value: "2.1%", trend: -0.5, trendInverted: true },
]
```

Avec `trendInverted: true`, une tendance negative s'affiche en vert.

## Notes

- **Server Component** par defaut — pas de `"use client"` necessaire.
- Le responsive est automatique : 1 colonne sur mobile, 2 sur tablette, `columns` sur desktop.
- Les tendances sont affichees avec `TrendingUp` (vert) ou `TrendingDown` (rouge) de Lucide.
