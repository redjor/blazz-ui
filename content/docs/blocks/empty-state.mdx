---
title: EmptyState
description: Etat vide avec icone, message et bouton d'action.
---

## Import

```tsx
import { EmptyState } from "@/components/blocks/empty-state"
```

## Utilisation

```tsx
import { Users, Plus } from "lucide-react"

<EmptyState
  icon={Users}
  title="Aucun contact"
  description="Ajoutez votre premier contact pour commencer."
  action={{
    label: "Ajouter un contact",
    href: "/contacts/new",
    icon: Plus,
  }}
/>
```

## Props

### `EmptyStateProps`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `LucideIcon` | - | Icone affichee au centre |
| `title` | `string` | **requis** | Message principal |
| `description` | `string` | - | Message secondaire |
| `action` | `EmptyStateAction` | - | Bouton d'action |
| `className` | `string` | - | Classes CSS additionnelles |

### `EmptyStateAction`

| Prop | Type | Description |
|------|------|-------------|
| `label` | `string` | Texte du bouton |
| `href` | `string` | Lien (rend un `<Link>`) |
| `onClick` | `() => void` | Handler de clic (rend un `<button>`) |
| `icon` | `LucideIcon` | Icone a gauche du label |

## Variantes

### Minimal — titre seul

```tsx
<EmptyState title="Aucun resultat" />
```

### Avec action onClick

```tsx
<EmptyState
  icon={FileText}
  title="Aucun document"
  description="Importez vos premiers documents."
  action={{
    label: "Importer",
    onClick: () => setImportOpen(true),
    icon: Upload,
  }}
/>
```

### Utilisation dans DataGrid

Le composant s'integre naturellement comme `emptyState` du DataGrid :

```tsx
<DataGrid
  columns={columns}
  data={[]}
  totalCount={0}
  emptyState={
    <EmptyState
      icon={Users}
      title="Aucun contact"
      action={{ label: "Ajouter", href: "/contacts/new" }}
    />
  }
/>
```

## Notes

- **Server Component** — pas de `"use client"` necessaire si l'action utilise `href`.
- Si l'action a un `onClick`, le parent doit etre un Client Component.
- L'icone est entouree d'un cercle `bg-muted` pour plus de visibilite.
