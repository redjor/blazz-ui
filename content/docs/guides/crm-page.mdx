## Objectif

Creer une page `/contacts` complete avec :
- En-tete avec breadcrumbs et bouton d'ajout
- Statistiques en haut de page
- Table de donnees avec tri, filtres et pagination
- Etat vide quand il n'y a pas de donnees

## Etape 1 : Creer la route

```tsx
// app/(dashboard)/contacts/page.tsx
import { fetchContacts, fetchContactStats } from "@/lib/actions/contacts"
import { ContactsPageClient } from "./contacts-page-client"

interface PageProps {
  searchParams: Promise<Record<string, string>>
}

export default async function ContactsPage({ searchParams }: PageProps) {
  const params = await searchParams
  const [{ data, totalCount }, stats] = await Promise.all([
    fetchContacts(params),
    fetchContactStats(),
  ])

  return (
    <ContactsPageClient
      data={data}
      totalCount={totalCount}
      stats={stats}
    />
  )
}
```

Le Server Component fetche les donnees et passe au Client Component pour l'interactivite.

## Etape 2 : Composant client

```tsx
// app/(dashboard)/contacts/contacts-page-client.tsx
"use client"

import { Plus } from "lucide-react"
import { PageHeader } from "@/components/blocks/page-header"
import { StatsGrid } from "@/components/blocks/stats-grid"
import { DataGrid } from "@/components/blocks/data-grid"
import { EmptyState } from "@/components/blocks/empty-state"
import type { Contact } from "@/types"

interface Props {
  data: Contact[]
  totalCount: number
  stats: StatItem[]
}

export function ContactsPageClient({ data, totalCount, stats }: Props) {
  return (
    <div className="space-y-6 p-6">
      <PageHeader
        title="Contacts"
        description={`${totalCount} contacts au total`}
        breadcrumbs={[
          { label: "CRM", href: "/dashboard" },
          { label: "Contacts" },
        ]}
        actions={[
          { label: "Ajouter", icon: Plus, href: "/contacts/new" },
        ]}
      />

      <StatsGrid stats={stats} columns={4} />

      <DataGrid
        columns={contactColumns}
        data={data}
        totalCount={totalCount}
        selectable
        emptyState={
          <EmptyState
            title="Aucun contact"
            description="Ajoutez votre premier contact."
            action={{ label: "Ajouter", href: "/contacts/new", icon: Plus }}
          />
        }
      />
    </div>
  )
}
```

## Etape 3 : Definir les colonnes

```tsx
import type { ColumnDef } from "@/components/blocks/data-grid"

const contactColumns: ColumnDef<Contact>[] = [
  {
    id: "name",
    header: "Nom",
    sortable: true,
    cell: (row) => (
      <div className="flex items-center gap-2">
        <Avatar className="size-6">
          <AvatarFallback>{row.name[0]}</AvatarFallback>
        </Avatar>
        <span className="font-medium">{row.name}</span>
      </div>
    ),
  },
  {
    id: "email",
    header: "Email",
    cell: (row) => row.email,
  },
  {
    id: "company",
    header: "Societe",
    sortable: true,
    cell: (row) => row.company?.name ?? "—",
  },
  {
    id: "status",
    header: "Statut",
    cell: (row) => (
      <Badge variant={row.status === "active" ? "default" : "outline"}>
        {row.status}
      </Badge>
    ),
  },
]
```

## Etape 4 : Server Actions

```tsx
// lib/actions/contacts.ts
"use server"

import { db } from "@/lib/db"

export async function fetchContacts(params: Record<string, string>) {
  const page = Number(params.page) || 1
  const pageSize = 25
  const sortField = params.sortField || "name"
  const sortDirection = params.sortDirection === "desc" ? "desc" : "asc"

  const [data, totalCount] = await Promise.all([
    db.contact.findMany({
      take: pageSize,
      skip: (page - 1) * pageSize,
      orderBy: { [sortField]: sortDirection },
      include: { company: { select: { name: true } } },
    }),
    db.contact.count(),
  ])

  return { data, totalCount }
}
```

## Resultat

Vous avez maintenant une page CRM complete avec :

1. **PageHeader** — navigation et actions
2. **StatsGrid** — KPIs en un coup d'oeil
3. **DataGrid** — table triable et paginee
4. **EmptyState** — UX soignee quand il n'y a pas de donnees
5. **Server Actions** — donnees fetched cote serveur

Pour aller plus loin, remplacez `DataGrid` par `DataTable` avec un preset CRM pour avoir les filtres avances et les vues sauvegardees.
