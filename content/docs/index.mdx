## Bienvenue

**Pro UI Kit** est un kit de composants React/Next.js concu pour construire des applications professionnelles data-heavy. Il cible les Lead Techs qui veulent vibe coder des apps internes au lieu de payer Salesforce ou SAP.

## Ce que vous trouverez ici

### Blocks

Les **block components** sont des composants metier de haut niveau, prets a l'emploi pour construire des pages CRM, dashboards et interfaces data-rich. Chaque block gere ses 4 etats : loading, empty, error et success.

[Explorer les blocks](/docs/blocks)

### Data Table

Le systeme **DataTable** est un composant avance construit sur TanStack Table avec des presets CRM, des column factories et un systeme de filtres.

[Decouvrir DataTable](/docs/data-table)

### Guides

Des guides pas-a-pas pour construire des pages completes avec le kit.

[Lire les guides](/docs/guides)

## Stack technique

- **Next.js 16** avec React 19
- **TypeScript strict** — tout est type
- **Tailwind CSS v4** avec design tokens custom
- **shadcn/ui** etendu — primitives UI solides
- **TanStack Table** — tables data-heavy
- **react-hook-form + zod** — formulaires valides
- **Prisma + PostgreSQL** — persistence
